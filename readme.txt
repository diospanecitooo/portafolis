En la práctica 1, realizada en el contexto de la Unidad Formativa 2, se trata de configurar un servidor LDAP en un entorno Debian, así como gestionar usuarios y grupos. Para lograrlo, se utilizan diversas tecnologías y comandos del sistema operativo, como la instalación y configuración del servidor OpenLDAP, la modificación de contraseñas mediante comandos como slappasswd, la creación de usuarios y organizaciones unitarias (OUs) con ldapadd, la visualización de la configuración con slapcat, entre otros.
