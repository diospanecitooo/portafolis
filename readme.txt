En la práctica 1, realizada en el contexto de la Unidad Formativa 2, se trata de configurar un servidor LDAP en un entorno Debian, así como gestionar usuarios y grupos. Para lograrlo, se utilizan diversas tecnologías y comandos del sistema operativo, como la instalación y configuración del servidor OpenLDAP, la modificación de contraseñas mediante comandos como slappasswd, la creación de usuarios y organizaciones unitarias (OUs) con ldapadd, la visualización de la configuración con slapcat, entre otros.

En este trabajo se han realizado una serie de tareas relacionadas con la configuración de servidores y la gestión de nodos utilizando Ansible en un entorno Debian. Se han seguido pasos detallados para la instalación y configuración de SSH, generación de claves SSH, configuración de los nodos, creación de estructuras de archivos y carpetas para Ansible, así como la creación y ejecución de playbooks para instalar y desinstalar aplicaciones como SL y Audacity en los nodos. Se han utilizado tecnologías como SSH para la comunicación segura entre servidores y nodos, Ansible para la automatización de tareas de configuración y gestión, y se han empleado scripts YAML para definir las tareas a realizar en los nodos.
